# Temperature Converter

A simple Python application for converting temperatures between Celsius and Fahrenheit.

## Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Easy to use

## Requirements

- Python 3.x

## Run

```bash
python converter.py
```

## Example

```
25°C = 77°F
68°F = 20°C
```
